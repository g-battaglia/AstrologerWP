AstrologerWP is a powerful astrology plugin that allows you to integrate advanced astrology calculations and chart generation into your WordPress site.

== Description ==

AstrologerWP is a WordPress plugin that connects to the [Astrologer API](https://rapidapi.com/gbattaglia/api/astrologer) to provide astrology calculations, natal charts, and zodiac analysis.

**Features:**
- Generate natal charts dynamically
- Multiple astrology house systems and zodiac types
- Customizable themes for astrology charts
- Supports multiple languages
- Easy to integrate via shortcode

== Installation ==

1. Upload the `astrologerwp` folder to `/wp-content/plugins/`
2. Activate the plugin through the "Plugins" menu in WordPress
3. Configure your API key in the plugin settings (Settings -> Astrologer API)
4. Configure your Geonames username in the plugin settings (Settings -> Astrologer API)
4. Use the `[astrologer_chart]` shortcode to display charts

== Frequently Asked Questions ==

= How do I get an API key? =
You can get an API key by subscribing to the [Astrologer API](https://rapidapi.com/gbattaglia/api/astrologer/pricing).

= Does this plugin require any dependencies? =
Yes, it requires an active API key from the Astrologer API.

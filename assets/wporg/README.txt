WordPress.org Plugin Directory Assets
======================================

This directory contains placeholder assets for the WordPress.org plugin
directory listing. Before submitting to WP.org, replace these SVG
placeholders with final PNG files at the required dimensions.

Required files (PNG format for WP.org):

  Icon (plugin listing thumbnail):
    icon-128x128.png   - Standard resolution
    icon-256x256.png   - High-DPI / Retina

  Banner (plugin page header):
    banner-772x250.png  - Standard resolution
    banner-1544x500.png - High-DPI / Retina

The SVG files in this directory serve as design references. Export them
to PNG at the correct pixel dimensions before uploading to the WP.org
SVN `assets/` directory.

See: https://developer.wordpress.org/plugins/wordpress-org/plugin-assets/
